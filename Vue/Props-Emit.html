props-emit 及父链

在组件中，父组件和子组件的作用域是相互独立的，但是组件之间协同工作那么又是可能会相互调用对方的数据或者说相互通信，vue中通过props向下传递数据，通过events向上传递消息。

 

1、Props

1.1、数据传递

props由父组件传递数据给子组件，那么所传递数据需要在子组件的标签上进行声明，如下子组件标签名为my-com，其中的属性message和：msg，message传递的数据是普通字符串，使用v-bind:msg绑定的是动态数据，可以拿到父组件的数据，并且随着数据变化而变化。

拿到数据我们还需要在子组件中声明属性props，如props:['message','msg']。这样就可以在子组件内使用传递过来的数据了。

xxxxxxxxxx
<div id="com">
    <my-com message="hello"></my-com>
    <my-com :msg="data"></my-com>
</div>
​
<script>
  Vue.component('my-com',{
      props:['message','msg'],
      template:"<h2>是的{{message}}{{msg}}</h2>"
  });
    new Vue({
        el:"#com",
        data:{
            data:"World !!!"
        }
    });
​
</script>

未使用v-bind的传递的数据类型只能是普通字符串数据，使用了v-bind绑定的数据是动态数据，可以有多种类型，如object，number等。

1.2、props验证

上面组件中的props我们使用的是数组来承载数据，其实我们还可以使用对象形式。如下：

xxxxxxxxxx
props:{  
        message:String, 
        msg:String 
},

prop 是一个对象而不是字符串数组时，组件可以为 props 指定验证要求。如果未指定验证要求，Vue 会发出警告。以上数据都是字符串数据。具体数据验证格式如下：

其中required指定是否必须传入数据，default为默认值。

  props: {
    // 基础类型检测 （`null` 意思是任何类型都可以）
    propA: Number,
    // 多种类型
    propB: [String, Number],
    // 必传且是字符串
    propC: {
      type: String,
      required: true
    },
    // 数字，有默认值
    propD: {
      type: Number,
      default: 100
    },
    // 数组／对象的默认值应当由一个工厂函数返回
    propE: {
      type: Object,
      default: function () {
        return { message: 'hello' }
      }
    },
    // 自定义验证函数
    propF: {
      validator: function (value) {
        return value > 10
      }
    }
  }

2、自定义事件

父组件是使用 props 传递数据给子组件，但如果子组件要把数据传递回去，应该怎样做？那就是自定义事件！

每个 Vue 实例都实现了事件接口：

  • 使用 $on(eventName) 监听事件
  • 使用 $emit(eventName) 触发事件
2.1、中央总线bus传递事件

每个实例都可以作为bus，但是为了简洁，建议新建一个实例专门作为bus.

<div id="com">
    <my-com :msg="message"></my-com>
    {{message}}
</div>
​
<script>
    var com_1={
        template:"<div>大家好，我是Component" +
            "<h1><button @click='getEmit'>点击我{{name}}{{msg}}</button></h1>" +
            "</div>",
        props:{
            'msg':String
        },
        data(){
            return{
                name:"Jack"
            }
        },
        methods:{
            getEmit(){
                bus.$emit('Es',"这是子组件向父组件传递消息了。。。");
                this.name=this.msg
            }
        }
    };
    var bus=new Vue();
    new Vue({
        el:"#com",
        data:{
          message:'Hello'
        },
        components:{
            'my-com':com_1
        },
        mounted:function () {
            var _this=this;
            bus.$on('Es',function (msg) {
                _this.message=msg
            })
        }
    });
​
</script>
2.2、使用v-on绑定自定义事件

父组件可以在使用子组件的地方直接用 v-on 来监听子组件触发的事件。

xxxxxxxxxx
<div id="counter-event-example">
  <p>{{ total }}</p>
  <button-counter v-on:increment="incrementTotal"></button-counter>
  <button-counter v-on:increment="incrementTotal"></button-counter>
</div>
​
.............................................................
​
Vue.component('button-counter', {
  template: '<button v-on:click="increment">{{ counter }}</button>',
  data: function () {
    return {
      counter: 0
    }
  },
  methods: {
    increment: function () {
      this.counter += 1
      this.$emit('increment')
    }
  },
})
new Vue({
  el: '#counter-event-example',
  data: {
    total: 0
  },
  methods: {
    incrementTotal: function () {
      this.total += 1
    }
  }
})
​

 

3、父链

使用$parent可以访问到父组件的数据，同理使用$children可以访问到所有子组件的数据。

<div id="com">
    <my-com></my-com>
</div>
​
<script>
    var com_1={
        template:"<div>大家好，我是Component" +
            "<h1><button @click='getEmit'>点击我</button></h1>" +
            "</div>",
        methods:{
            getEmit(){
                alert(this.$parent.message)
            }
        }
    };
    new Vue({
        el:"#com",
        data:{
            message:'Hello'
        },
        components:{
            'my-com':com_1
        }
    });
​
</script>
